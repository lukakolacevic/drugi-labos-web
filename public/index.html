Demonstracija sigurnosnih ranjivosti web-aplikacija

Cross-site scripting (XSS)

Upute:

Ovo je demonstracija XSS (Cross-site scripting) ranjivosti. Kada je ranjivost uključena, unos korisnika se ne sanitizira i omogućava izvršavanje JavaScript koda.

Kako testirati:

  • Unesite poruku sa JavaScript kodom, npr: <img src=x onerror="alert('XSS: ' + document.cookie)">
  • Primjer s klikom: Unesite: <div onclick="alert('XSS napad izvršen! Cookie: ' + document.cookie)" style="cursor:pointer;color:blue;padding:10px;border:1px solid red;">Klikni na mene za besplatan mobitel</div> - Nakon što umetnete ovu poruku, pogledajte u listu poruka i kliknite na plavi okvir s crvenim borderom kako biste izvršili XSS napad
  • Kada je ranjivost isključena, HTML znakovi će biti escapirani i kod se neće izvršiti

Loša kontrola pristupa (Broken Access Control)

Upute:

Ova ranjivost omogućava korisnicima pristup resursima koji im ne bi trebali biti dostupni.

Kako testirati:

  • Prijavite se kao obični korisnik (username: user, password: user123)
  • Kada je ranjivost uključena, obični korisnik može pristupiti admin stranici i vidjeti sve korisnike
  • Kada je ranjivost isključena, samo admin (username: admin, password: admin123) može pristupiti admin stranici
